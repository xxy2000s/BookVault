    718. 最长重复子数组¶

    给两个整数数组 nums1 和 nums2 ，返回 两个数组中 **公共的* 、长度最长的子数组的长度* 。

    Follow UP:

    • dp优化到一维 ✅
    • 二分+字符串哈希
    • 滑动窗口
    • 输出子串（子数组）
    //dp 明确一个二维的矩阵，然后降维优化
class Solution {
    public int findLength(int[] nums1, int[] nums2) {
        int l1 = nums1.length;
        int l2 = nums2.length;
        int dp[] = new int[l2+1];
        int res = 0;
        //原本dp是i行j列，优化后只存当前遍历的右方和左上方，从而降维
        for(int i=1;i<=l1;i++){
            //因为先运算后面不影响前面，所以必须逆序遍历
            for(int j=l2;j>=1;j--){
                //这里的dp[j]就是原来的dp[i][j]
                if(nums1[i-1] == nums2[j-1]) dp[j] = dp[j-1]+1;
                //由于要局部完全匹配，不匹配则重置。不同于子序列
                else dp[j] = 0;
                res = Math.max(res, dp[j]);
            }
        }
        return res;
    }
}

    1143. 最长公共子序列¶

    给定两个字符串 text1 和 text2，返回这两个字符串的最长 公共子序列 的长度。如果不存在 公共子序列 ，返回 0 。

    Follow UP:

    • dp优化到一维✅
    • 要求输出子序列✅
    //不同于滚动数组，起始就是取决于左，上，斜上三个地方。加一个遍历存斜上即可优化为一维
//二维的话转移方程显而易见是1.dp[i][j] = dp[i-1][j-1]+1; 2.dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1])  寻求最大子串匹配
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int l1 = text1.length();
        int l2 = text2.length();
        int dp[] = new int[l2+1];
        for(int i=1;i<=l1;i++){
            //pre就相当左上角的dp[i-1][j-1]
            int pre = 0;
            for(int j=1;j<=l2;j++){
                //dp[j]要被改变,所以暂存作为左上角的dp[i-1][j-1]并之后得赋予给pre。
                int tmp = dp[j];
                //dp[j]这时候被更新计算了，所以不一定还是之前的dp[i-1][j-1]
                if(text1.charAt(i-1) == text2.charAt(j-1)) dp[j] = pre+1;
                else dp[j] = Math.max(dp[j], dp[j-1]);
                //暂存左上角dp[i-1][j-1]供下一轮循环使用
                pre = tmp;
            }
        }
        return dp[l2];
    }
}
